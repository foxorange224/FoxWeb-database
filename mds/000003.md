Para poder instalar VLC en una Distro de Linux, tendras que instalarlo en tu gestor de paquetes de tal distro, no es tan dificil:

## Arch Linux:
sudo pacman -Syu vlc vlc-plugins-all

## Debian o Ubuntu:
sudo apt install vlc
